## Анализ базы фильмов MovieLens

MovieLens — это веб-система рекомендаций и виртуальное сообщество,  которое рекомендует пользователям фильмы для просмотра на основе их  предпочтений с помощью совместной фильтрации оценок и рецензий членов  сообщества. Она содержит около 11 миллионов оценок примерно 8500  фильмов. MovieLens был создан в 1997 году GroupLens Research,  исследовательской лабораторией факультета компьютерных наук и инженерии  Университета Миннесоты, с целью сбора исследовательских данных и  персонализированных рекомендаций.  

Цель данного проекта - анализ данных из сокращенной версии MovieLens.  Основные задачи:  

1) Написать модуль, содержащий все необходимые методы, включая парсинг csv-файлов и сайтов, поиск и статистический анализ фильмов, пользовательских оценок и т.д., а также тесты для всех используемых методов. Модуль на языке Python: `src/movielens_analysis.py`  
2) Подготовить отчет об анализе базы MovieLens с использованием только методов, содержащихся в нашем модуле. Отчет в формате Jupyter Notebook: `src/movielens_report.ipynb`  

Примеры классов и методов приведены в `code-samples`, csv-файлы с данными и кратким описанием - в папке `datasets`.   

- Класс Movies отвечает за анализ данных из movies.csv: распределение фильмов по годам выхода и жанрам.  

- Класс Ratings отвечает за анализ данных из ratings.csv: распределение рейтингов по фильмам и пользователям.  

- Класс Tags отвечает за анализ данных из tags.csv: анализ пользовательских тегов фильмов.  
- Класс Links отвечает за загрузку и анализ данных из базы IMDb: бюджет, режиссер, IMDb-рейтинг и др.  
- Класс Tests содержит тесты для всех используемых методов.
